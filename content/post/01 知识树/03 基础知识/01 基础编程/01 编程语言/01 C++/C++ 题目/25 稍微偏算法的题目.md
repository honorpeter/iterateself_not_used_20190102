#### Q 60 :

题目：

请问func(2012,2102)的结果是：

```
int fuc(int m,int n){
    if(m%n == 0)
        return n；
    else
        return fuc(n,   m%n)；
}
```

答案：

2

解答：
考察辗转相除法。
1. 辗转相除法求两个数的最大公约数。




#### Q 68 :

题目：
有以下程序:
```
#include < stdio.h >
int main(){
    char a[5][10] ={"one", "two", "three", "four", "five"};
    int i, j;
    char t;
    for (i = 0; i < 2; i++){
        for(j = i + 1; j < 5; j++ ){
            if(a[i][0] > a[j][0]){
                t = a[i][0];
                a[i][0] = a[j][0];
                a[j][0] = t;
            }
        }
    }
    puts(a[1]);
}
```

答案：
fwo

解答：
1. 对五个字符串首字母进行冒泡排序。
