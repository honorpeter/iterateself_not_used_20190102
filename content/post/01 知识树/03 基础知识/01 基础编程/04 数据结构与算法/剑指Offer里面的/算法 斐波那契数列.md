## 题目：


打印出斐波那契数列。

f(n)=f(n-1)+f(n-2)


## 解答：


普通的方法：


    def fib(n):
        if n < 1:
            raise ValueError
        elif n <= 2:
            return 1
        else:
            return fib(n - 1) + fib(n - 2)


评价：

在fib(100)的时候已经很慢了，因为重复的计算太多
