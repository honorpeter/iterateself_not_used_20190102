Python 'ascii' codec can't decode byte 0xe9 in position 0 ordinal not in range(128)

2018-06-11 97 words 1 min read times read

Contents

  • 缘由
  • 解决办法
  • REF

缘由

在使用 Python2.7 读取文件的时候，提示说： ‘ascii’ codec can’t decode byte 0xe9 in position 0: ordinal not in range(128)”

解决办法

要添上：

import sys
default_encoding = 'utf-8'
if sys.getdefaultencoding() != default_encoding:
    reload(sys)
    sys.setdefaultencoding(default_encoding)
